Certificados

Gestión Masiva de Certificados

El sistema permite administrar la emisión y consulta de certificados de forma masiva a través de dos funcionalidades principales:

  1. Consulta de Certificados Emitidos: Esta función permite visualizar y gestionar el registro histórico de certificados emitidos para todos los matriculados en el sistema.

  1. Generación Masiva de Certificados: Esta funcionalidad permite crear nuevos certificados de forma masiva para un grupo seleccionado de matriculados.

Certificados Emitidos

La Consulta de Certificados Emitidos permite visualizar y gestionar el registro histórico de certificados emitidos para todos los matriculados en el sistema. Además permite realizar la descarga de un listado en formato .csv.

A partir de un filtro de búsqueda avanzados por:

  • Datos del matriculado
  • Tipo de certificado
  • Organización receptora
  • Fecha de emisión
  • Etc.

Se obtiene el resultado de la búsqueda y es posible exportarla en un listado haciendo click en el botón "Descargar CSV".



Generación Masiva de Certificados

Generación Masiva de Certificados Esta funcionalidad permite crear nuevos certificados de forma masiva para un grupo seleccionado de matriculados. El proceso de generación es el siguiente:

  1. Selección de matriculados mediante filtros de búsqueda
  2. Especificación de los datos del certificado:
    1. Tipo de certificado a generar
    2. Organización receptora

El sistema procesará la creación individual de certificados para cada matriculado seleccionado, generando documentos únicos y manteniendo el registro específico para cada profesional. Esta gestión masiva facilita la administración eficiente de certificados para múltiples matriculados, asegurando la precisión y el control en la emisión de documentación oficial.
