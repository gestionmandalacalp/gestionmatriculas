Documentos

Gestión de Documentos de un Matriculado

En esta sección se listan todos los documentos asociados a la matrícula del profesional, como:

  • Títulos académicos
  • Certificaciones
  • Constancias
  • Antecedentes
  • Etc.

El usuario que gestiona la documentación del Matriculado deberá verificar la integridad y validez de la documentación presentada. Esto permitirá contar con un registro histórico de los documentos que respaldan la matrícula e Identificar cualquier cambio o actualización en la documentación del profesional.
Tener acceso a estos documentos es crucial para asegurar el cumplimiento de los requisitos normativos y garantizar la legitimidad y correcta manutención de la información de la Matrícula del profesional.

Nuevo Documento

Para agregar un nuevo documento en la sección de Documentos:

Al hacer clic en el botón "Nuevo Documento", se presentará un formulario para ingresar los detalles relevantes:

Título: El usuario proporciona un título descriptivo para el nuevo documento que se va a agregar.

Descripción: El usuario puede incluir una breve descripción textual o explicación del documento.

Tipo de Documento: El usuario selecciona el tipo de documento apropiado de una lista predefinida, como Título, Certificación, Licencia, etc.

Cargar Archivo: Luego, el usuario puede navegar en sus archivos locales y adjuntar el archivo del documento, como un PDF o imagen, para asociarlo al registro de matrícula.

Una vez que se proporciona toda la información requerida y cargado el archivo del documento, el usuario puede guardar la información. Esto agregará el documento a la lista completa que se muestra en la sección de Documentos, haciéndolo parte del legajo del profesional.

Vista, Descarga y Eliminación de Documento

A continuación se brinda una explicación breve de las acciones de visualización, descarga y eliminación de documentos en el sistema de gestión de matrículas.

  • Visualización: Al hacer clic en un documento de la lista, se abrirá una vista previa del archivo, permitiendo al usuario revisar su contenido sin necesidad de descargar.
  • Descarga: Junto a cada documento, habrá una opción para descargarlo. Al seleccionar esta opción, el sistema generará y proporcionará el archivo para que el usuario pueda guardarlo localmente.
  • Eliminación: Si el usuario tiene los permisos adecuados, podrá eliminar un documento de la lista. Esto eliminará de manera permanente el archivo del registro de matrícula. Por este motivo se requiere absoluta responsabilidad al ejecutar esta operación.
