Plantillas de Certificados

Gestión de Plantillas

Este módulo permite la gestión y configuración de las plantillas utilizadas por el sistema para la generación de diferentes tipos de certificados oficiales. El sistema permite crear y administrar múltiples tipos de certificados, como Certificado de Matrícula, Certificado de Habilitación, Certificado de Libre Deuda, entre otros.
Para cada tipo de certificado, se establece un formato y diseño predefinido que será utilizado durante la emisión.
Las plantillas incluyen campos y etiquetas dinámicas (tags) que serán reemplazados con la información específica del matriculado durante la generación del certificado.
Estos tags permiten incorporar datos como nombre, número de documento, fecha de matriculación, título profesional, entre otros.
El administrador puede personalizar el diseño, distribución de elementos y estilos de la plantilla según los requisitos de cada certificado.
Las plantillas actualizadas se aplican automáticamente a la emisión de nuevos certificados.
Esta funcionalidad asegura la consistencia y estandarización en la generación de certificados oficiales, facilitando la personalización según los requisitos particulares de cada tipo de documento.

Creación de Plantilla de Certificado

Para generar una nueva plantilla de certificado, el usuario debe seguir estos pasos:

  1. Hacer clic en el botón "Crear" ubicado en la sección correspondiente.
  2. En el formulario de creación, el usuario deberá proporcionar la siguiente información:
    1. Título de la Plantilla: Nombre descriptivo que identifica el tipo de certificado.
    2. Descripción: Breve explicación sobre el propósito y uso de la plantilla.
    3. En el campo de "Contenido", el usuario podrá incluir los tags dinámicos provistos por el sistema. Estos tags serán reemplazados automáticamente con la información específica del matriculado durante la generación del certificado. Algunos ejemplos de tags disponibles son:[Nombre][Número de Documento][Fecha de Matriculación]

Una vez completados todos los datos requeridos, el usuario puede guardar los cambios para crear la nueva plantilla de certificado.
Esta plantilla estará entonces disponible para su selección y uso en el proceso de emisión de certificados oficiales para los profesionales matriculados.
